в данном уркое мы с вами подготовим всё самое необходимое для разработки сайта.

Среда разрабатки

Для того, чтобы нам писать код, нам понадобится среда разрабокти. Существует двольно много различных сред разрабатки, но мы посоветуем начать с классики и это VS code ссылка для установки

Настройка VS code

Первым делом нужно немного настроить VS code для работы. Все настройки вы можете посмотреть в интернете. Я же посоветую всего паррочку настроек это: русский язык и размер шрифта кода. Чтобы поставить русский язык нам нужно зайти в расширения(это такие 4 квадратика на левой ленте внизу) и написать rus. Нам выдаст расширение для VS code, которое принсоит в него великий и могучий. Нажимаем и скачиваем, после установки перезапускаем VS code.

Далее для того, чтобы поменять размер шрифта текста вам нужно: нажать на шесеренку в нижнем левом углу->параметры->font-zise->ставим любое значение, которое вам нравится
